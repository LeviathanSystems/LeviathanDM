# Plugin Development

LeviathanDM supports status bar widget plugins written in C++.

## Plugin API

Plugins use a simple API to create custom widgets for the status bar.

See the [Plugin Development Guide](https://github.com/LeviathanSystems/LeviathanDM/blob/master/plugins/PLUGIN_DEV_GUIDE.md) for details.

## Example Plugins

- **Battery Widget** - Display battery status using UPower
- **Clock Widget** - Display current time
- **Tags Widget** - Show workspace tags

Check the [plugins directory](https://github.com/LeviathanSystems/LeviathanDM/tree/master/plugins) for examples.
